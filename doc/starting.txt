## Architecture

<p>The&nbsp;<strong>OPC UA Solution .NET</strong> is a set of .NET assemblies that supports writing of server and client applications for the OPC UA
specification.&nbsp;<strong>OPC UA Solution .NET</strong> consists of two developer products, one for&nbsp;<a href=
"https://technosoftware.com/opc-ua-client-net/">Client Development</a> and one for <a href="https://technosoftware.com/opc-ua-server-net/">Server
Development</a>. Both use the same OPC UA Core assembly which provides the communication stack and common functionality used in servers and clients
applications:</p>

<p>&nbsp;</p>

\image html OPCUANETArchitecture.png

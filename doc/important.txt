## NuGet packages

<p>The&nbsp;<strong>OPC UA Solution .NET</strong> is delivered as a set of&nbsp;<a href="https://www.nuget.org/profiles/technosoftware" target=
"_blank"><u><font color="#0066CC">NuGet</font></u></a> packages. Depending on the targeted .NET Framework you have the following&nbsp;<a href=
"https://www.nuget.org/profiles/technosoftware" target="_blank"><u><font color="#0066CC">NuGet</font></u></a> packages available.</p>

### .NET 7.0, .NET 6.0 or&nbsp;.NET 4.8

<ol>
    <li><a href="https://www.nuget.org/packages/Technosoftware.UaSolution.UaClient/" target="_blank">Technosoftware.UaSolution.UaClient</a><br />
    The OPC UA Client .NET offers a fast and easy access to the OPC UA Client technology. Develop OPC UA compliant UA Clients with C#.</li>

    <li><a href="https://www.nuget.org/packages/Technosoftware.UaSolution.UaServer/" target="_blank">Technosoftware.UaSolution.UaServer</a><br />
    The OPC UA Server .NET offers a fast and easy access to the OPC Unified Architecture (UA) technology. Develop OPC UA&nbsp;compliant Servers with C#.</li>

    <li><a href="https://www.nuget.org/packages/Technosoftware.UaSolution.UaPubSub/" target="_blank"><u><font color=
    "#0066CC">Technosoftware.UaSolution.UaPubSub</font></u></a><br />
    The OPC UA PubSub .NET offers a fast and easy access to the OPC Unified Architecture (UA) technology. Develop OPC UA compliant Publisher and Subscriber
    with C#.</li>
</ol><b>Dependencies:</b>

<ol>
    <li><a href="https://www.nuget.org/packages/Technosoftware.UaSolution.UaConfiguration/" target="_blank">Technosoftware.UaSolution.UaConfiguration</a><br />
    The OPC UA Client &amp; Server .NET uses a common set of configuration options which is the content of this package.</li>

    <li><a href="https://www.nuget.org/packages/Technosoftware.UaSolution.UaCore/" target="_blank">Technosoftware.UaSolution.UaCore</a><br />
    Based on the OPC UA .NET Standard Stack from the OPC Foundation.</li>

    <li><a href="https://www.nuget.org/packages/Technosoftware.UaSolution.UaBindings.Https/" target=
    "_blank">Technosoftware.UaSolution.UaBindings.Https</a><br />
    If your OPC UA Client or OPC UA Server needs support of HTTPS then you need to reference also this package.</li>
</ol>

<p>.NET 7.0 and .NET 6.0 allows you to develop applications that run on all common platforms available today, including Linux, macOS and Windows 8.1/10
(including embedded/IoT editions) without requiring platform-specific modifications.</p>

<p>One of the reference implementations inside this project has been certified for compliance through an OPC Foundation Certification Test Lab to prove its
high quality. Fixes and enhancements since the certification process have been tested and verified for compliance using the Compliance Test Tool (CTT)
V1.04.</p>

<p>Furthermore, cloud applications and services (such as ASP.NET, DNX, Azure Websites, Azure Webjobs, Azure Nano Server and Azure Service Fabric) are also
supported.</p>

## Features Included

<ul>
    <li>Fully ported Core OPC UA Stack.</li>

    <li>X.509 Certificate support for client and server authentication.</li>

    <li>SHA-2 support (up to SHA512) including security profile Basic256Sha256, Aes128Sha256RsaOaep and Aes256Sha256RsaPss for configurations with high
    security needs.</li>

    <li>Anonymous, username and X.509 certificate user authentication.</li>

    <li>UA-TCP &amp; HTTPS transports (client and server).</li>

    <li>Reverse Connect for the UA-TCP transport (client and server).</li>

    <li>Folder &amp; OS-level (X509Store) Certificate Stores with Global Discovery Server and Server Push support.</li>
</ul>
